**Key Directories**
- `app/`: Controllers, models, middleware, helpers, and console kernel. Sources: `app/Http/Controllers/*`, `app/Models/*`, `app/Http/Kernel.php`, `app/Console/Kernel.php`.
- `routes/`: Web routes and console routes. Sources: `routes/web.php`, `routes/console.php`.
- `resources/views/`: Blade templates for all UI pages and layout. Sources: `resources/views/**/*`.
- `resources/js/` and `resources/css/`: Vite entrypoints, Tailwind config, and JS utilities. Sources: `resources/js/app.js`, `resources/js/app.jsx`, `resources/css/app.css`.
- `config/`: App configuration including database, auth, cache, queue, sessions, mail, logging. Sources: `config/*.php`.
- `database/`: Migrations and seeders. Sources: `database/migrations/*`, `database/seeders/*`.
- `public/`: Web root and static assets. Source: `public/` directory.
- `storage/`: App storage, logs, cache, and backup archives. Source: `storage/` directory.
- `tests/`: Default example tests. Sources: `tests/Feature/ExampleTest.php`, `tests/Unit/ExampleTest.php`.
